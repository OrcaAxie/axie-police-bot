# axie-police-bot
Telegram bot to manage Axie Infinity telegram group (https://t.me/axieinfinity)

### Installation
`npm install node-telegram-bot-api`

Modify config.js, add the telegram token got from @BotFather

### Run the bot

`node index.js`
